# selenium-sample
Project created to expose my suggestion of feature to the Selenium project.

## pre-requisites

| Name    | Required |
|---------|----------|
| Docker  | true     |
| Java 21 | true     |


## running the tests
1. clone this repo
```
git clone https://github.com/ClaudenirFreitas/selenium-sample.git
```

2. run the tests:
```
./gradlew clean test
```

3. open the test report:
```
open build/reports/tests/test/index.html
```
